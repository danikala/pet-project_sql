# SQL Pet-project
    В данном проекте создана таблица model со следующей структурой
![schema](schema.png)

    В процессе я изучал все способы работы с СУБД PostgreSQL начиная с базового создания таблиц и изучения языка запросов.

    Part 1: как создавать SQL-запросы, используя внутренние подзапросы в предложениях FROM и SELECT, а также фильтровать данные по диапазону дат

    Part 2: фундаментальные навыки работы с SQL (работа с наборами и операторами set, создание подзапросов, сортировка по нескольким столбцам, а также типы объединения таблиц)

    Part 3: понимание реляционной алгебры, использование различных операций JOINs, написание запросов с помощью CTEs, обработка NULL значений и фильтрация/сортировка данных

    Part 4: как извлекать данные из базы данных с помощью сложных SQL-запросов и изменять их с помощью языка управления данными (DML)

    Part 5: как создавать и использовать представления для фильтрации данных, создания временных диапазонов, определения пропущенных дат, работы с установленными операциями, расчета скидок и управления материализованными представлениями, включая их обновление и удаление

    Part 6: создание индексов базы данных

    Part 7: интегрировать новую бизнес-функцию в модель данных

    Part 8: как использовать определенные OLAP-конструкции для извлечения информации из данных

    Part 9: знакомство с уровнями изоляции транзакций и их значением в управлении базами данных

    Part 10: создание функций (SQL/PLpgSQL) и триггеров для реализации бизнес-логики на уровне СУБД, разработка систем аудита изменений данных и оптимизиация запросов.
